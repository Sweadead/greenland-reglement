Règlement officiel

GREENLAND RUST

En rejoignant le serveur, chaque joueur accepte l'intégralité du présent règlement. L'ignorance d'une règle ne constitue jamais une excuse.

I. Taille des équipes

La limite est fixée à 3 joueurs maximum.

Une équipe comprend toute personne autorisée sur le Tool Cupboard (TC), les cadenas, les tourelles automatiques, ou possédant un lit ou un sac dans la base principale.

Exemples interdits
  • Farmer à quatre.
  • Faire un raid à quatre.
  • Défendre une base avec une quatrième personne.
  • Construire avec une quatrième personne.
  • Faire intervenir un ancien membre de l'équipe.
  • Quitter l'équipe pendant un raid afin de revenir quelques minutes plus tard.

Toute tentative de contourner cette règle sera considérée comme du teaming.

II. Alliances

Les alliances sont interdites.

  • Pactes de non-agression permanents.
  • Équipes amies permanentes.
  • Défenses communes.
  • Raids organisés ensemble.
  • Partage régulier de ressources.

Les collaborations ponctuelles restent tolérées uniquement lorsqu'elles résultent naturellement du jeu et qu'elles ne procurent pas un avantage durable.

Le staff reste seul juge de la différence entre une interaction ponctuelle et une alliance.

III. Échanges

Les échanges entre équipes sont autorisés, mais doivent rester ponctuels.

  • Via Shop Front.
  • Via Vending Machine.
  • Lors d'un échange rapide.

Les prêts permanents de ressources ou d'équipements sont interdits.

IV. Bases

Chaque équipe doit posséder sa propre base.

Il est interdit :
  • D'héberger une autre équipe.
  • De construire une base pour une autre équipe.
  • De partager une compound.
  • De partager des défenses.
  • De partager un TC.

Les bases satellites destinées uniquement à contourner les règles pourront être supprimées.

V. PvP

Le PvP est libre. Le kill on sight (KOS) est autorisé.

Le camping est autorisé tant qu'il utilise uniquement les mécaniques normales du jeu.

L'utilisation de bugs ou d'exploits est interdite.

VI. Raids

Les raids sont autorisés à toute heure.

Pendant un raid :
  • Aucune autre équipe ne peut intervenir volontairement.
  • Aucune aide extérieure ne peut être demandée.
  • Aucune défense extérieure organisée n'est autorisée.

Les contre-raids naturels restent autorisés si l'équipe n'a aucun lien avec les participants.

VII. Le grief

Le grief est strictement interdit sur Greenland Rust.

Est considéré comme du grief tout comportement visant principalement à empêcher un joueur ou une équipe de continuer à jouer normalement après une action PvP ou un raid.

Actions interdites :
  • Condamner volontairement une base.
  • Remplacer un TC dans le seul but de bloquer définitivement les propriétaires.
  • Poser des murs, portes, fondations ou constructions destinés à empêcher durablement la récupération d'une base.
  • Détruire ou modifier une base après un raid sans nécessité de gameplay, uniquement pour nuire.
  • Utiliser une mécanique, un bug ou un glitch pour rendre une base inutilisable.
Après un raid :

Les assaillants peuvent piller la base et sécuriser temporairement leur extraction. Toute action visant à condamner, bloquer ou rendre définitivement la base inutilisable est considérée comme du grief et sera sanctionnée.

VIII. Fair-play

Sont strictement interdits :

  • Cheats.
  • Scripts.
  • Macros.
  • Logiciels tiers.
  • ESP.
  • Aimbot.
  • Recoil script.
  • Exploitation volontaire de bugs.

Toute découverte de bug doit être signalée.

IX. Comportement

Aucune tolérance concernant :

  • Racisme.
  • Homophobie.
  • Discrimination.
  • Harcèlement.
  • Menaces réelles.

Sont également interdits : spam, flood, publicité, usurpation d'identité d'un membre du staff.

X. Signalements

Toute plainte devra contenir une vidéo, une capture d'écran ou un combatlog si nécessaire.

Les accusations sans preuve seront ignorées. Les faux signalements volontaires pourront être sanctionnés.

XI. Staff

Le staff représente l'autorité du serveur. Les décisions sont finales.

Toute tentative de mentir, cacher des informations, falsifier des preuves ou contourner une sanction pourra entraîner une aggravation de la sanction.
